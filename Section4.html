css structure

selector : { property : value ;}

h1 { color : purple ;} an example that sets all the h1 purple

How to write them

We have three options

  1. Directly inline in html elements : by using the style attribute like h1 style: backgroundcolor : blue; but its not a good way
  2. By using the Style element in the head element and write for what element you want what things to happen but its not a good way either like : style h1{ color : blue;}
  3. Writing a new file are write our css there so we can link it to the our html files and for link it we have to use the link element in head link href="cssfile" rel ="stylesheet"
