BOX MODEL

The idea that everything in css is a box(Search it on google)

Each box has some pieces (content,padding,border,margin,width & height)

Width & Height

They control the content box

Border

The name is very self explainatory these below are some important properties

The border-width :property lets you change the width of the border

The border-style : controls the line style (dashed,solid,none ...)

The border-color : controls the color of the border

Border can be added from outside and inside by changing the box-sizing property to border-box

you can change the first three property by using the border attribute

Padding

Padding is the space between the content and the border

You can use the padding property or the padding-left and right and ...

Margin

Is the space outside the border and is the space between this element and other elements

Display Property

some elements are inline and some are block,Display property is very important these below are some of its values

  • Inline : makes element behave as inline,in inline elements width & height are ignored margin and padding push elements horizontali only
  • Block : make element behave as a block,in block elements every box model behavoir is respected
  • inline-block : behaves like inline but respects the box model behavior like block elements
