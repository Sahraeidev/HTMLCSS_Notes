Flexbox

A new part of css and it is seires a properties that we use to layout items and sort them and its highly useful

For using the flex box we must set display property to flex

Flex direction : by default it uses flex-direction row and it puts the properties in a row but you can change it

  • row
  • row-reverse
  • column
  • column-reverse

Justify-Content : how the content is diterbuted across the main axis by default is flex-start

  • flex-start
  • flex-end
  • center
  • space between
  • space around

Flex-wrap : how its wraps content in a small space(cross the main axis(x axis)),wrap and wrap-reverse and nowrap

Align-items : align items in cross axis(y axis) and has some similar values ( baseline value aligns them in a order so thier lines match)

align-content : (works when flex wrap is wrap or reverse-wrap) controls the space between the rows

align-self : is very similar to align-content but its a property for a single element in a nav for example

Flex-basis : the initial size(height or width depending on the main axis) of the element

Flex-grow : controls the amount of space that an element should take up

flex-shrink : when there is not space this determent that how much space they take up(0 means it will not shrink)

flex : it sets the 3 properties on top of this flex : (Flex-grow,flex-shrink,flex-basis) (Read MDN)
