Selector

Is used for using it on diffrent element and giving all of them the same css properties

we have multiple types of selectors like :

  • Element selector: select all of the elements of that type and gives them the same property changes / You can also select multiple elements and writing thier name next to each other
  • ID Selector: you can use the id attibute on a single element and style it diffrently from the others and use it like this on css #idname {color : black;}
  • Class selector: is one of the most commonly used its used for multiple elements to be style similary and its used by using the class attribute on the elements that you want and on the css file use .classname{ color : blue};
  • Descendant selector: [element] [nested elements inside the element before that you want to change] {} this type of selector is used to style the elements that are nested inside another element
  • Adjacent selector : h1 + p {color : red}; selects the paragraphs that come after h1
  • Direct child : div > li {}selects the li that are direct children of the div
  • Attribute selector : selects all the elements with a specific attribute input[type="text"]{}

If you want to give a property to everything you can use the * selector

Pseudo classes

Keywords added to a selector that specifies a special state of the selected element

selector:peusdoclass {color : blue;}

  • :hover : when the element is beign hovered
  • :active : when something is active (when you click on a link but dont release it)
  • :checked : when something is checked (radio buttons and checks and selects)
  • :nth-of-type(NUMBER(3) or NUMBERn(Selecting every n number like 3n selects every 3 number like 3 , 6, 9)) : styles the NUMBER(like third) of that element
