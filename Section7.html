Position

Its a confusing property and super useful and sets how a element is positioned in the document

  • Static : it is the default option,
  • Reletive : is going to keep the element in the flow of the document but we can ofset it reletive to itself using top left right bottom properties
  • Absolute : the element is removed from the normal document flow and no space is created from the element and when you change its position it changes reletive to its parent or a positioned element
  • Fixed : it has a fixed position
