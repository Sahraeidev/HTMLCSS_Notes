Bootstrap Section 1

What is a Framework?

A framework is a collection of pre-written code that helps you build projects more easily and quickly.

How to Include Bootstrap

There are multiple ways to include Bootstrap in your project:

  1. Download it and include the files in your project locally.
  2. Use a CDN (Content Delivery Network): a hosted version you can include by copying and pasting a link into your <head> element.
  3. Include it via JavaScript (optional, for components that require JS behavior).
